# ⚽ Copa 2026 — Sportsbook Calendar

Calendário interno do time de Sportsbook para gerenciar eventos, promoções e comunicação entre setores durante a Copa do Mundo 2026.

## 🚀 Como rodar

**Pré-requisito:** Node.js 18+

```bash
# Instalar dependências (primeira vez)
npm install

# Rodar localmente
npm run dev
# Acesse: http://localhost:5173

# Build de produção
npm run build
```

## 📋 Funcionalidades

- **Calendário por dia** — 104 jogos reais da fase de grupos
- **Eventos por jogo** — tipo, título, regras, prioridade e setores
- **Resumo geral** — visão consolidada com estatísticas
- **Filtro por setor** — Marketing, Atendimento, Operações, CRM, Trading, Compliance, TI
- **Impressão** — exporta como PDF

## 🛠 Tecnologias
React 18 · Vite 8 · Tailwind CSS v4 · Framer Motion · Lucide React · date-fns
